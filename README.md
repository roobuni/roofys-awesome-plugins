# Roofy's Awesome Plugins

my awesome plugins

## Plugins Included

### LetMeSee
locally caches messages for channels where you don't have permission to read message history, dumb permission anyway.
- clean up old messages
- per channel limits
- search
- never caches dms

### XLAccountSwitcher
Switch between Discord accounts using stored tokens.
- mass import for **those** types of people
- account management
- quick switching. (right click your profile in any chat)

### AutoDeleteWelcome
auto-deletes your own welcome messages when you join a server

### ChannelLogger
keeps deleted channels visible so you can still view cached messages before they're gone
- caches messages as you browse
- preserves deleted channels
- right-click to actually delete

### MakeshiftEveryone
adds a button in server context menu that pings all members - ghetto @everyone
- works even without mention perms
- fetches all member ids
